# UPBGE Architecture repository

A repository for new UPBGE architecture documents.

This is a repository for design or architecture documents for missing features, missing modules or features that don't work as expected. Mainly, these documents are a start step before any code is developed.

They can be created with own knowledge or IA or a mix of both. It is better to note it in the document.

Please, use Markdown format.
